RAMS CALCULATION ENGINE
RAMS metrics depend on:
job cards
fleet km
failure categories
5.1 MDBF
Mean Distance Between Failures
MDBF = Total Fleet KM / Total Failures
Example
Fleet KM = 1,000,000
Failures = 50

MDBF = 20000 km
5.2 MDBCF
Mean Distance Between Critical Failures
Critical failures include

delay
withdrawal
deboarding
fail dispatch
Formula
MDBCF = Fleet KM / Critical Failures
5.3 MTTR
Mean Time To Repair
MTTR = Total Repair Time / Total Failures
Repair time
Job Close Time – Failure Time
5.4 Failure Rate
Failure Rate = Failures / Fleet KM
5.5 Availability
Availability =
MTBF / (MTBF + MTTR)
Where
MTBF = MDBF / Avg Train Speed
5.6 Pattern Failure Detection
Identify repeated failures.
Example query

SELECT component,
COUNT(*)
FROM job_cards
GROUP BY component
ORDER BY count DESC

door motor failures
HVAC failures
brake valve failures

5 RAMS CALCULATION ENGINE
Based on RAMS Plan definitions.
MDBF definition

MDBCF definition
MDBCF = Fleet KM / Component Failures
MTTR
MTTR = Sum repair time / failures
rams_engine.py
from sqlalchemy.orm import Session
from models import JobCard

def calculate_rams(db: Session):

    failures = db.query(JobCard).count()

    total_km = sum(
        j.km_reading for j in db.query(JobCard).all()
    )

    total_repair = sum(
        j.duration_hr for j in db.query(JobCard).all()
    )

    MDBF = total_km / failures if failures else 0

    MTTR = total_repair / failures if failures else 0

    return {
        "MDBF": MDBF,
        "MTTR": MTTR
    }

Pattern failure definition (RAMS plan):

≥20% fleet occurrence
def pattern_failure(db):

    from sqlalchemy import func

    result = db.query(
        JobCard.component,
        func.count(JobCard.id)
    ).group_by(
        JobCard.component
    ).all()

    patterns = []

    for comp, count in result:

        if count >= 3:
            patterns.append(
                {
                    "component": comp,
                    "failures": count
                }
            )

    return patterns
